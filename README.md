# Rjanidor

### Introduction

My (**@arjansunar**) Personal PDE.

### Installation

- Backup current Neovim files
    ```bash
    # required
    mv ~/.config/nvim{,.bak}

    # optional but recommended
    mv ~/.local/share/nvim{,.bak}
    mv ~/.local/state/nvim{,.bak}
    mv ~/.cache/nvim{,.bak}
    ```

- Install Rjditor
    ```
    git clone https://github.com/LazyVim/starter ~/.config/nvim
    ```
